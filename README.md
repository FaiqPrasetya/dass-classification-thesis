# DASS Classification
This is a machine learning classification which uses Random Forest and SVM algorithm. It contains multiple processes, namely:
- Data Exploration
  - Data Visualization
- Data Preprocessing
  - Empty data and outlier handling with Imputation
  - Label Creation
  - Feature Engineering
- Model Implementation and Testing
- Evaluation

## Why are there multiple notebooks?
The processes I've mentioned above were done in separate notebooks, which is why there are multiple of them in this project.

- Original dataset from OpenPsychometrics is "data_export.csv"
- Data exploration is done in "skripsi_visualization" and "skripsi_visusalization_after_feature_engineering"
- Data preprocessing is done in "skripsi_dataexploration_imputation_v2" and "skripsi_feature_engineering"
- Model implemenetation and evaluation is done in "skripsi_model_final"
